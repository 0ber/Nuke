ImageManager
  • ImageManager

    The ImageManager class and related classes provide methods for loading, processing, caching and preheating images.

    ImageManager is also a pipeline that loads images using injectable dependencies, which makes it highly customizable. See https://github.com/kean/Nuke#design for more info.

ImageLoader
  • ImageLoader

    Performs loading of images for the image tasks using objects conforming to ImageDataLoading, ImageDecoding and ImageProcessing protocols. Works in conjunction with the ImageManager.

    • Provides a transparent loading, decoding and processing with a single completion signal
    • Reuses data tasks for equivalent image tasks
