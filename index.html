Easy to Use

Simple API. Zero configuration required. Comprehensive usage guide included.

Fast

Performant, asynchronous and thread-safe. The entire framework operates in the background.

Extensible

Nuke is a pipeline with injectable dependencies. Features Alamofire and FLAnimatedImage plugins.

Features

  • Hassle-free image loading into image views (and other targets)
  • Two cache layers including auto-purging memory cache
  • Image transformations
  • Automated preheating (prefetching)
  • Pipeline with injectable dependencies
  • and FLAnimatedImage plugins

Animated Images

Nuke features FLAnimatedImage plugin that allows you to load and display animated GIFs with smooth scrolling performance and low memory footprint.
