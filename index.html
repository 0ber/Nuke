Easy to Use

User-friendly API. Zero configuration required. Beautiful playground and comprehensive usage guide included.

Fast

Performant, asynchronous and thread-safe. Even creation of data tasks is performed on the background thread.

Extensible

Nuke is a pipeline with injectable dependencies. It features Alamofire and FLAnimatedImage plugins.

Features

Loading

  • Uses NSURLSession with HTTP/2 support
  • Install Alamofire plugin or integrate your own networking code in minutes
  • A single data task is used for multiple equivalent requests
  • Full featured extensions for UI components

Caching

  • Doesn't reinvent caching, relies on HTTP cache in URL Loading System
  • Two cache layers including auto-purging memory cache for processed images
  • Intellegent control over memory cache

Processing

  • Create, compose and apply image filters
  • Simple Core Image integration
  • Background image decompression and scaling in a single step
  • Resize loaded images to fit displayed size

Advanced

  • Nuke is a pipeline with injectable dependencies, everything is customizable
  • Automated preheating of images close to the viewport
  • Add UI extensions to your custom classes using traits

Animated Images

Nuke features FLAnimatedImage plugin that allows you to load and display animated GIFs with a silky smooth scrolling performance and extremely low memory footprint.

Installation

Supported platforms: iOS 8.0+ / watchOS 2.0+ / OS X 10.9+ / tvOS 9.0+

CocoaPods

To install Nuke add a dependency to your Podfile:

{% highlight ruby %} # source 'https://github.com/CocoaPods/Specs.git' # use_frameworks! # platform :ios, "8.0" / :watchos, "2.0" / :osx, "10.9" / :tvos, "9.0" pod "Nuke" pod "Nuke-Alamofire-Plugin" # optional pod "Nuke-AnimatedImage-Plugin" # optional {% endhighlight %}

Carthage

To install Nuke add a dependency to your Cartfile:

{% highlight ruby %} github "kean/Nuke" github "kean/Nuke-Alamofire-Plugin" # optional github "kean/Nuke-AnimatedImage-Plugin" # optional {% endhighlight %}
