Easy to Use

Simple and expressive API. Zero configuration required. Comprehensive usage guide included.

Fast

Performant, asynchronous and thread-safe. Handles heavy workloads with ease.

Extensible

Nuke is a pipeline with injectable dependencies. Alamofire, Gifu, Toucan plugins are available.

Features

Nuke pulls together stable, mature libraries from Swift ecosystem into simple, lightweight package that lets you focus on getting things done.

  • Hassle-free image loading into image views and other targets
  • Two cache layers including LRU memory cache
  • Image transformations
  • Freedom to use networking, caching libraries of your choice
  • Alamofire, Gifu, Toucan plugins
  • Automated prefetching with Preheat library
  • Performant, handles heavy workloads with ease

Animated Images

Nuke features Gifu and FLAnimatedImage (deprecated) plugin that allows you to load and display animated GIFs with smooth scrolling performance and low memory footprint.
