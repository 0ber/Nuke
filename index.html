Loading, processing, caching and preheating images

{% include twitter_share_button.html %}
Easy to Use

Simple API. Zero configuration required. Beautiful playground and comprehensive usage guide included.

Fast

Performant, asynchronous and thread-safe. Even creation of data tasks is performed on the background thread.

Extensible

Nuke is a pipeline with injectable dependencies. Features Alamofire and FLAnimatedImage plugins.

Features

Loading

  • Install Alamofire plugin or integrate your own networking code in minutes
  • Extensions for UI components
  • Deduplicate equivalent requests

Caching

  • Two cache layers including auto-purging memory cache for processed images
  • Protocol for integrating any third-party caching library

Processing

  • Create, compose and apply image filters
  • Simple Core Image integration
  • Background image decompression

Advanced

  • Image preheating (precaching)
  • Implement custom data loaders, caches, processors, decoders, etc
  • Add UI extensions to your classes using traits

Animated Images

Nuke features FLAnimatedImage plugin that allows you to load and display animated GIFs with a silky smooth scrolling performance and extremely low memory footprint.

Installation

Supported platforms: iOS 8.0+ / watchOS 2.0+ / OS X 10.10+ / tvOS 9.0+

CocoaPods

To install Nuke add a dependency to your Podfile:

{% highlight ruby %} # source 'https://github.com/CocoaPods/Specs.git' # use_frameworks! # platform :ios, "8.0" / :watchos, "2.0" / :osx, "10.10" / :tvos, "9.0" pod "Nuke" pod "Nuke-Alamofire-Plugin" # optional pod "Nuke-AnimatedImage-Plugin" # optional {% endhighlight %}

Carthage

To install Nuke add a dependency to your Cartfile:

{% highlight ruby %} github "kean/Nuke" github "kean/Nuke-Alamofire-Plugin" # optional github "kean/Nuke-AnimatedImage-Plugin" # optional {% endhighlight %}
